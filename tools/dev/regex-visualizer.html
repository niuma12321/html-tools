正则表达式参考

字符类

. 任意字符
\d 数字 [0-9]
\w 单词字符
\s 空白字符
[abc] 字符集
[^abc] 否定字符集

量词

* 0 或更多
+ 1 或更多
? 0 或 1
{n} 恰好 n 次
{n,} n 次或更多
{n,m} n 到 m 次

边界

^ 字符串开始
$ 字符串结束
\b 单词边界
\B 非单词边界

分组

(abc) 捕获组
(?:abc) 非捕获组
(?<name>abc) 命名捕获
a|b 或
\1 反向引用
