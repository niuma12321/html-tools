🍪 Cookie 解析器

解析和分析 HTTP Cookie，检查安全属性设置

📝 输入 Cookie

点击下方按钮读取当前页面的 Cookie（仅能读取非 HttpOnly 的 Cookie）

💡 Cookie 安全属性说明

  • Secure - 仅通过 HTTPS 传输，防止中间人窃取
  • HttpOnly - 禁止 JavaScript 访问，防止 XSS 窃取
  • SameSite - 控制跨站请求时是否发送 Cookie
    • Strict - 完全禁止跨站发送
    • Lax - 允许安全的顶级导航
    • None - 允许跨站发送（需配合 Secure）
  • Domain - Cookie 有效的域名范围
  • Path - Cookie 有效的路径范围
  • Max-Age / Expires - Cookie 过期时间
